This is a sample paragraph you can select and edit. Click the pencil button in the top-right corner to open the editor popup, then use the "Elements" tab on the left edge to add new elements. Click "Save" (or Ctrl+S) to write your changes back to this file on disk — closing the popup reloads the page with what you saved. Shift+click to select multiple elements at once and batch-edit their styles or delete them together; the breadcrumb bar at the bottom shows the selected element's ancestry.
